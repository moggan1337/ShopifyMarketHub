## вңЁ Features

### Vendor Management
- рҹ“Ҡ **Vendor Dashboard** - Sales analytics, earnings, performance metrics
- рҹҸӘ **Vendor Registration** - Application flow with approval workflow
- вӯҗ **Vendor Ratings** - Review system with dispute handling
- рҹ“қ **Vendor Profiles** - Custom branding, banners, bio

### Commerce
- рҹ’° **Commission Engine** - Flexible rules, tiered commissions, categories
- рҹ’і **Split Payments** - Automatic vendor payouts with Stripe Connect
- рҹ“Ұ **Shipping Rules** - Per-vendor, flat rate, weight-based
- рҹӣ’ **Cart Consolidation** - Multi-vendor cart with single checkout
- рҹ“ң **Invoices** - Auto-generated vendor invoices

### Operations
- вҡ–пёҸ **Dispute Resolution** - Ticket system with mediation
- рҹ“Ҡ **Admin Analytics** - Platform-wide metrics, vendor rankings
- рҹ”” **Notifications** - Email, SMS, in-app notifications
- рҹ“ұ **PWA Ready** - Mobile-optimized vendor app

## рҹ“Ұ Installation

```bash
git clone https://github.com/moggan1337/ShopifyMarketHub.git
cd ShopifyMarketHub
npm install
cp .env.example .env
npm run db:migrate
npm run dev
```

## рҹҡҖ Quick Start

```bash
# Configure Stripe Connect in .env
STRIPE_SECRET_KEY=sk_live_xxxxx
STRIPE_WEBHOOK_SECRET=whsec_xxxxx

# Set commission rules
COMMISSION_RATE=0.15  # 15% platform fee

# Start development
npm run dev
```
